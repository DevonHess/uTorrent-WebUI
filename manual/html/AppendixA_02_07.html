µTorrent User Manual > Appendix A: The µTorrent Interface > Preferences

Transfer Cap

The transfer cap is used to control the amount of data µTorrent will transfer over a specified period of time.

Enable Transfer Cap will enable the transfer cap if checked.

Cap Settings

  • The Bandwidth Cap field sets the amount of data to use as the transfer cap for the selected period.
  • The Time Period dropdown menu sets the number of days to use as the transfer cap period.
  • The Limit Type option sets the transfer direction(s) to count as part of the transfer cap.

Usage History

This displays textual statistics pertaining to the amount of data transferred for the selected time period. The "Reset History" button below resets the collected statistics.
